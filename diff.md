現在開いているファイルの PR での差分を解説してください。

以下の手順で実行:

1. **ベースブランチの特定**
   - `gh pr view --json baseRefName -q .baseRefName` でPRのベースブランチを取得
   - PRがない場合は develop をデフォルトとする

2. **リネーム検出**
   - `git diff -M50 <ベースブランチ>...HEAD --summary` を実行
   - 対象ファイルがリネーム（移動）されているか確認
   - リネームの場合は旧パスを特定

3. **差分の取得**
   - リネームの場合: `git diff -M50 <ベースブランチ>...HEAD -- <旧パス> <新パス>` を実行
   - 通常の場合: `git diff <ベースブランチ>...HEAD -- <現在のファイルパス>` を実行
   - ファイルが新規追加の場合は全内容を表示

4. **差分の解説**
   - リネームの場合は移動元・移動先を明記
   - 変更箇所ごとに何が変わったか説明
   - 変更の意図・目的を推測して解説
   - コードの品質や潜在的な問題点があれば指摘

5. **まとめ**
   - 変更の全体像を簡潔に要約
