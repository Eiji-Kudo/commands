# 本日の目標生成 (Daily Goals Generator)

あなたは以下の情報を収集し、ユーザーの「今日の最優先事項 (MIT)」と「本日の主要目標」を提案してください。

## 情報収集

1. **割り当てられたGitHub Issues**を確認:
   ```bash
   gh issue list --assignee @me --state open --limit 10
   ```

2. **最近のGit Activity**を確認:
   - 直近のコミット履歴
   - 現在のブランチと作業状況
   - プルリクエストの状況

3. **未完了のPRレビュー**を確認:
   ```bash
   gh pr list --search "review-requested:@me"
   ```

## 出力フォーマット

収集した情報を基に、以下の形式で提案してください：

### 今日の最優先事項 (MIT: Most Important Thing)

- 会社/プロジェクトが前に進むために「これだけは絶対に終わらせる」と決めた、たった一つのこと
- 具体的で、完了条件が明確なもの
- 例: 「認証機能のPR #123 をマージ可能な状態にする」

### 本日の主要目標 (Today's Goals)

- MITを補完する2〜3の主要なタスクや会議
- 箇条書きで簡潔に
- 具体的なアクションがわかるように動詞で書く
- 「〇〇について考える」ではなく「〇〇の仕様案をVer1.0としてまとめる」

## 注意事項

- 優先度が高いissue、期限が近いものを重視する
- 進行中のPRがあればそれを優先的に完了させることを提案する
- ユーザーが選択・調整できるよう、複数の選択肢を提示しても良い
