プロジェクトのCI チェックを実行し、問題があれば修正してください。

例えば以下の順序で実行:

1. **Lint チェック & 修正**
   - `bun run lint:check` または `npx biome check .` を実行
   - エラーがあれば修正

2. **Type チェック & 修正**
   - `bun run typecheck` または `npx tsc --noEmit` を実行
   - 型エラーがあれば修正

3. **テスト実行 & 修正**
   - `bun run test` を実行
   - 失敗したテストがあれば修正

各ステップで問題が見つかった場合は、修正後に再度該当チェックを実行して成功を確認してから次に進んでください。
すべてのチェックがパスするまで繰り返してください。
